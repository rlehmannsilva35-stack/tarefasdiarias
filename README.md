# Tarefas Diárias

Uma aplicação simples de lista de tarefas para acompanhar atividades diárias.

## Como usar

1. Abra `tarefas.html` no navegador.
2. Digite uma tarefa no campo e clique em **Adicionar**.
3. Marque a caixa para indicar tarefa concluída.
4. Use **Limpar concluídas** para remover tarefas finalizadas e **Marcar todas** para alternar o estado.

## Arquivos

- `tarefas.html` — interface da lista de tarefas
- `tarefas.css` — estilo visual
- `tarefas.js` — lógica de adicionar, marcar e salvar tarefas

## Observações

- As tarefas são salvas no `localStorage` do navegador.
- Não é necessário backend.
